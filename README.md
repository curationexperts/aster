# Aster
### GeoBlacklight for UCSB

## Running the application

1. Checkout the repo: 
  `git clone git@github.com:curationexperts/aster.git && cd aster`
1. Create a gemset for dependencies: 
  `rvm use --create 2.4.2@aster`
1. Install the gems: 
  `bundle install` 
1. Run the server: 
  `rake geoblacklight:server`
